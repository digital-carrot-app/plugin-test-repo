# Todoist

Create goals using data from your todoist tasks. This plugin allows you
to filter tasks using any [todoist filter](https://www.todoist.com/help/articles/introduction-to-filters-V98wIH).
With filters you can do things like set goals where you have to finish
all of your tasks that are:

- Due today
- Overdue
- High priority
- In a specific project
- And anything else you can think to filter!
